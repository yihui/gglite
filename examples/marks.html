Marks (Geometry Layers)

library(gglite)

Point Mark (Scatter Plot)

g2(mtcars, x = 'mpg', y = 'hp') |> mark_point()

Color by a variable

g2(mtcars, x = 'mpg', y = 'hp', color = 'wt', size = 5) |>
  mark_point(style = list(shape = 'point'))

Size by a variable

g2(mtcars, x = 'mpg', y = 'hp', size = 'wt') |> mark_point()

Shape by a variable

Note: in G2 v5, shape encoding in point marks may not always match between the legend and the plot. See G2 source and documentation for details.

g2(iris, x = 'Sepal.Width', y = 'Sepal.Length', color = 'Species',
    shape = 'Species') |>
  mark_point()

Custom style

g2(mtcars, x = 'mpg', y = 'hp', size = 10) |>
  mark_point(style = list(fill = 'lightyellow', stroke = 'steelblue', lineWidth = 2))

Line Mark

df = data.frame(x = 1:10, y = cumsum(rnorm(10)))
g2(df, x = 'x', y = 'y') |> mark_line()

Multi-series line

  x = rep(1:5, 2), y = c(3, 1, 4, 1, 5, 2, 7, 1, 8, 3),

g2(df, x = 'x', y = 'y', color = 'group') |> mark_line()

Path Mark

The path mark in G2 uses an SVG-like d channel to draw arbitrary shapes. It does not use x/y data coordinates like mark_line(). Use mark_line() to connect data points in order.

g2(df, x = 'x', y = 'y') |> mark_path()

A line mark connecting data points in order:

n = 100
t = seq(0, 4 * pi, length.out = n)
df = data.frame(x = t * cos(t), y = t * sin(t))
g2(df, x = 'x', y = 'y') |> mark_line()

Polygon Mark

The polygon mark draws filled polygons. Use list columns to store vertex coordinates for each polygon.

df = data.frame(group = c('triangle', 'square'))
df$x = list(c(0, 1, 0.5), c(2, 3, 3, 2))
df$y = list(c(0, 0, 1), c(0, 0, 1, 1))
g2(df, color = 'group') |>
  mark_polygon(encode = list(x = 'x', y = 'y'))

Link Mark

df = data.frame(

  x1 = c('B', 'C', 'D'), y1 = c(3, 1, 4)

Density Mark

The density mark requires a KDE transform that is not available in G2 v5. Use mark_heatmap() as an alternative for 2-D density visualizations.

g2(iris, x = 'Sepal.Width', y = 'Sepal.Length') |>
  mark_density(
    transform = list(list(type = 'kde')),
    style = list(fill = 'steelblue', fillOpacity = 0.5)
  )

Partition (Sunburst) Mark

The partition mark does not work correctly in G2 v5. Use mark_treemap() or mark_pack() as alternatives for hierarchical data.

tree_data = list(
  name = 'root', children = list(
    list(name = 'A', value = 10, children = list(
      list(name = 'A1', value = 5), list(name = 'A2', value = 5)
    )),
    list(name = 'B', value = 20)
  )
)
g2() |>
  mark_partition(
    data = list(value = tree_data),
    encode = list(value = 'value'),
    coordinate = list(type = 'theta', innerRadius = 0.3)
  )
